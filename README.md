## 🎯 Game Overview

**Eternity Tech** is an immersive interactive storytelling game that blends:
- **Ancient Sanskrit** and Vedic knowledge
- **Quantum computing** concepts
- **Interactive puzzles** and cipher challenges
- **3D visualization** using Three.js
- **Mobile-optimized** touch controls

### Story Summary
Krishna and Sakshi Patil, guided by mentor Parth, have created revolutionary technology that merges ancient Sanskrit knowledge with quantum computing. When a deadly attack puts their lives in danger, they must navigate through puzzles, ciphers, and obstacles to unlock the secrets of Vault B in Padmanabhaswamy Temple.

## 💻 Installation & Setup

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)

## 🕹️ How to Play

### Game Flow
1. **Loading Screen** → Game loads assets
2. **Backstory** → Read the story introduction (can skip)
3. **Main Menu** → Choose "New Journey" or "Continue"
4. **Chapters** → Progress through 12 chapters with dialogue and puzzles
5. **Completion** → Finish all chapters to see the ending

### Game States
- **Dialogue**: Read character conversations, press SPACE (or tap) to continue
- **Puzzle**: Solve interactive puzzles to progress
- **Playing**: Explore and interact with the 3D environment
- **Menu**: Navigate between chapters

## 🧩 All Puzzle Solutions

### CHAPTER 1, SCENE 2: Vedic OS Algorithm
**Puzzle Type**: Sanskrit Sequence (Arrange in Order)

**Correct Order:**
1. ॐ (Om - Sacred sound)
2. गणेशाय (Ganeshaya - To Ganesh)
3. नमः (Namah - Salutations)
4. सरस्वत्यै (Saraswatai - To Saraswati)
5. विद्या (Vidya - Knowledge)
6. दान (Dan - Gift)

**Translation**: "Om, salutations to Ganesh and Saraswati, the gift of knowledge"

---

### CHAPTER 2, SCENE 2: Decrypt Parth's Cipher
**Puzzle Type**: Rune Alignment (3 Rings)

**Correct Alignment:**
- **Ring 1**: सूर्य (Surya - Sun)
- **Ring 2**: चन्द्र (Chandra - Moon)
- **Ring 3**: मंगल (Mangal - Mars)

**Meaning**: "Sun-Moon-Mars" - represents planetary alignment in Vedic astronomy

**How to Solve**:
- Ring 1: Click to rotate until सूर्य aligns with top marker
- Ring 2: Click to rotate until चन्द्र aligns with top marker
- Ring 3: Click to rotate until मंगल aligns with top marker

### CHAPTER 5, SCENE 2: Vedic Star Chart

### CHAPTER 6, SCENE 2: Melody of Passage
**Puzzle Type**: Sanskrit Sequence (Musical Notes - Sargam)

**Correct Order** (Hindustani Music Scale):
1. **Sa** (सा) - First note (Shadja)
2. **Re** (रे) - Second note (Rishabh)
3. **Ga** (ग) - Third note (Gandhar)
4. **Ma** (म) - Fourth note (Madhyam)
5. **Pa** (प) - Fifth note (Pancham)
6. **Dha** (ध) - Sixth note (Dhaivat)
7. **Ni** (नी) - Seventh note (Nishad)

**Translation**: "Sa Re Ga Ma Pa Dha Ni" = Indian musical notes (Indian version of Do Re Mi Fa Sol La Ti)

**How to Solve**:
- This is the traditional Indian musical scale
- Arrange them in ascending order (Sa to Ni)

## 🐛 Debugging Guide

### Common Issues & Solutions

#### 1. **Game Won't Load**
**Problem**: Blank screen or loading indefinitely

**Solutions**:
- Check browser console (F12 → Console tab) for errors
- Ensure JavaScript is enabled
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser
- Check internet connection (for CDN libraries)

```javascript
// Test in browser console
console.log("Game loaded:", typeof EternityTechGame);
```

#### 2. **3D Scene Not Rendering**
**Problem**: Game interface shows but no 3D scene visible

**Solutions**:
- Check WebGL support: `console.log(!!document.createElement('canvas').getContext('webgl'))`
- Update graphics drivers
- Try hardware acceleration settings in browser
- Use a different browser

####  3. **Touch Controls Not Working**
**Problem**: Buttons don't respond to touch on mobile

**Solutions**:
- Ensure touch events are enabled in browser
- Check that buttons have minimum 48px height
- Try tapping harder (if tap-sensitive)
- Reload page
- Ensure browser isn't in zoom mode

#### 4. **Audio/Haptic Not Working**
**Problem**: No vibration feedback on Android

**Solutions**:
```javascript
// Test haptic feedback
navigator.vibrate([100, 50, 100]);
```
- Check device has vibration enabled
- Check app has vibration permission
- Some browsers don't support Vibration API

#### 5. **Puzzles Not Accepting Correct Answer**
**Problem**: Correct answer marked as wrong

**Solutions**:
- Check exact spelling and formatting
- Ensure all items are properly selected
- Clear puzzle and reset before retrying
- Check text content matches (use browser inspector)

#### 6. **Dialogue Continues Automatically**
**Problem**: Skipping required interaction

**Solutions**:
- Ensure `gameState` is 'dialogue' before continuing
- Check event listeners are properly attached
- Verify keyboard/touch events are firing

## ❓ Troubleshooting

### Game Progress Issues

**Q: How do I save my progress?**
A: Currently, progress resets on page reload. To add save functionality, implement LocalStorage:
```javascript
// Save progress
localStorage.setItem('eternity_chapter', this.currentChapter);
localStorage.setItem('eternity_scene', this.currentScene);

// Load progress
const chapter = localStorage.getItem('eternity_chapter') || 1;
const scene = localStorage.getItem('eternity_scene') || 1;
```

**Q: Can I skip chapters?**
A: Yes, you can use the browser console:
```javascript
game.startChapter(5); // Jump to chapter 5
```

**Q: How do I reset the game?**
A: Reload the page or use:
```javascript
game = new EternityTechGame(); // Restart
```

### Puzzle Issues

**Q: I selected the right answer but it says wrong**
A: Ensure:
- No extra spaces in text
- Correct spelling (especially Sanskrit)
- All required items selected
- Try Reset button first

**Q: Cipher wheel won't rotate**
A: Try:
- Click directly on the ring (not the center)
- Single click rotates one position
- On mobile, try swiping instead of tapping

**Q: Hidden puzzle solution not showing**
A: Use the "View Puzzle" button at the top-right to reopen any closed puzzle

### Mobile Issues

**Q: Text too small on my phone**
A: The game auto-scales. Try:
- Refresh page
- Rotate device
- Check if browser zoom is at 100% (not pinched smaller)

**Q: Touch doesn't work on certain buttons**
A: Make sure you're tapping the button itself, not outside it. Buttons need 48px tap area.

**Q: Haptic vibration not working**
A: Check:
- Device has vibration enabled
- App has vibration permissions
- Try Android browser instead of app browser

Edit the CSS variables in `<style>`:


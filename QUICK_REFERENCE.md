## 🐛 COMMON ISSUES

| Problem | Solution |
|---------|----------|
| Game won't load | Clear cache, try different browser |
| 3D scene missing | Check WebGL support |
| Touch not working | Enable JavaScript, reload page |
| Wrong answer marked wrong | Ensure exact spelling |
| Haptic not working | Check Android vibration permission |
| Text too small | Rotate device, adjust zoom to 100% |
| Buttons won't click | Ensure 48px+ touch target area |
| Game stuck | Try pressing ESC or reloading |

## 📱 MOBILE TIPS

✅ **Do This:**
- Use portrait mode for dialogue
- Use landscape for 3D scene viewing
- Tap center of buttons
- Swipe cipher wheels left/right
- Allow vibration permission

❌ **Avoid:**
- Pinch zooming (disabled)
- Right-clicking (no context menu)
- Double-tapping (causes zoom)
- Tapping outside buttons
- Rapid repeated clicks
